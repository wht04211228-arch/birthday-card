# AI + 3D 礼物盒商业级电子贺卡前端

## 新增功能
- 3D 礼物盒开箱动画：点击开盖，触发烟花、爱心雨、隐藏祝福。
- AI 祝福生成：根据姓名、关系、共同回忆和语气生成祝福。
- AI 情书生成：自动生成长篇情书，并同步到情书弹窗。
- AI 文案改写：把普通祝福改写成高级商业文案。
- AI 语音朗读：浏览器本地语音朗读祝福内容。

## 如何定制客户版本
1. 修改 `config.js`：姓名、署名、生日、祝福、照片路径。
2. 替换 `assets/photo1.jpg`、`photo2.jpg`、`photo3.jpg`。
3. 把背景音乐命名为 `music.mp3`，放到 `assets` 文件夹。
4. 本地双击 `index.html` 预览，或上传到 Vercel / GitHub Pages。

## 后续接真 AI API
当前 `script.js` 里的 `generateLocalAI()` 是前端模拟 AI，适合演示和接单。
真正上线时建议：
- 前端提交姓名、关系、回忆、语气。
- 后端 Node.js/Express 调用 OpenAI、通义千问或智谱 API。
- 返回生成好的祝福/情书给前端展示。
