# 商业级交互电子贺卡前端

## 新增交互功能
- 开场解锁仪式页
- 背景音乐按钮
- 烟花粒子动画
- 祝福弹幕
- 心愿瓶留言
- 照片点击放大
- 点击页面星光彩蛋
- 爱心雨彩蛋
- 祝福氛围切换
- 分享文案复制
- 客户预览文案生成

## 如何定制客户版本
1. 修改 `config.js`：姓名、署名、生日、情书、祝福语。
2. 替换 `assets/photo1.jpg`、`photo2.jpg`、`photo3.jpg`。
3. 放入音乐文件 `assets/music.mp3`。
4. 上传到 Vercel / GitHub Pages 即可访问。
